# HTML标签 -- 按字母顺序排序

> ** ~~删除线~~ 表示不支持HTML5**

| 标签 | 描述 |
| :------------: | :------------: |
| `<!—…—>`  | 注释  |
| `<!DOCTYPE>`  | 文档类型  |
| `<a>`  |  一个超链接 |
|  `<abbr>` | 定义缩写或首字母缩写  |
| ~~`<acronym>`~~  | 首字母缩略词（使用`<abbr>`代替）  |
| `<address>`  |  文档作者/所有者的联系信息 |
|  ~~`<applet>`~~ | 嵌入式小程序（使用`<embed>`或`<object>`代替） |
| `<area>`  | 图像映射内的区域  |
| `<article>`  | 文章  |
| `<aside>`  | 除页面内容之外的内容  |
| `<audio>`  | 嵌入的音频内容  |
| `<b>`  | 粗体文本  |
| `<base>`  | 指定文档中所有相对URL的基本URL/目标  |
| ~~`<basefont>`~~  |  为文档中的所有文本指定默认颜色、大小和字体（使用CSS代替）|
| `<bdi>`  | 隔离文本的一部分，该部分的格式可能与该部分之外的其他文本的格式不同  |
| `<bdo>`  | 覆盖当前文本方向  |
| ~~`<big>`~~  | 大号文本 （使用CSS代替）  |
| `<blockquote>`  | 引用自其他来源的部分  |
| `<body>`  |  文档的主体 |
| `<br>`  | 单个换行符  |
| `<button>`  |  一个可点击按钮 |
|  `<canvas>` | 用于通过脚本(通常是JavaScript)动态地绘制图形  |
| `<caption>`  |  表标题  |
| ~~`<center>`~~ | 居中文本 （使用CSS代替）  |
| `<cite>`  | 作品的标题  |
|`<code>`   | 计算机代码文本  |
| `<col>`  | 为`<colgroup>`元素中的每一列指定列属性  |
| `<colgroup>`  | 指定要进行格式化的表中一个或多个列的组  |
| `<command>`  | 定义命令按钮，比如单选按钮、复选框或按钮  |
|  `<data>` | 添加给定内容的机器可读翻译  |
| `<datalist>`  |  为输入控件指定预定义选项的列表 |
| `<dd>`  |  在描述列表中定义术语的描述/值 |
| `<del>`  | 从文档中删除的文本  |
|  `<details>` | 用户可以查看或隐藏的其他详细信息  |
|  `<dfn>` |  指定将在内容中定义的术语 |
| `<dialog>`  |  一个对话框或窗口 |
| ~~`<dir>`~~  | 目录列表，使用“< ul >”代替 |
| `<div>`  | 文档中的一个部分  |
| `<dl>`  | 描述列表  |
| `<dt>`  | 在描述列表中定义术语/名称  |
|  `<em>` | 强调文本  |
| `<embed>`  | 为外部应用程序定义容器  |
| `<fieldset>`  | 将表单中的相关元素分组  |
| `<figcaption>`  | `<figure> `元素的标题  |
|  `<figure>` | 指定自包含的内容  |
|  ~~`<font>`~~ |  定义字体、颜色和文本大小（使用CSS代替） |
|`<footer>`   |  文档或部分的页脚   |
|`<form>`   | HTML表单  |
| ~~`<frame>`~~  | 框架集的窗口或框架  |
|  ~~`<frameset>`~~ | 一组框架  |
|`<h1>` to `<h6>`   | HTML标题  |
| `<head>`  | 包含文档的元数据/信息  |
|`<header>`   | 文档或部分的标题   |
| `<hr>`  | 水平线 |
|  `<html>` | HTML 文档 |
|`<i>`   | 斜体字 |
|`<iframe>`   | 内联框架 |
| `<img>`  | 图像 |
|   `<input>` | 输入控件 |
|`<ins>`   | 已插入到文档中的文本 |
|  `<kbd>`  | 键盘输入文本 |
| `<label>`  | 为`<input>`元素定义一个标签  |
| `<legend>`  | `<fieldset> `元素的标题  |
| `<li>`  | 列表的项目 |
| `<link>`  | 文档与外部资源的关系（最常用于链接到样式表） |
| `<main>`  | 文档的主要内容  |
| `<map>`  | 定义图像映射  |
| `<mark>`  | 定义标记/突出显示的文本 （在需要突出显示文本时使用 `<em>` 标签） |
| `<menu>` | 菜单列表 |
| `<meta>`  | 有关 HTML 文档的元数据 |
| `<meter>`  | 度量衡 （仅用于已知最大和最小值的度量）  |
| `<nav>`  | 导航链接 |
| ~~`<noframes>`~~  | 针对不支持框架的用户的替代内容 |
| `<noscript>`  | 针对不支持客户端脚本的用户的替代内容 |
| `<object>`  | 内嵌对象 |
|`<ol>`  | 有序列表 |
| `<optgroup>`  | 选择列表中相关选项的组合 |
| `<option>`  | 下拉列表中的选项 |
| `<output>`  | 不同类型的输出 |
| `<p>`  | 段落 |
| `<param>`  |  定义对象的参数 |
| `<picture>`  | 为多个图片资源定义一个容器 |
| `<pre>`  | 预格式化文本 |
| `<progress>`  | 运行中的进度（进程） |
| `<q>`  | 简短的引用 |
|  `<rp>` | 定义在不支持 ruby 注释的浏览器中显示的内容  |
| `<rt>`  |定义字符的解释/发音（中文注音或字符）   |
| `<ruby>`  | 定义ruby注释（中文注音或字符）  |
| `<s>`  | 不再正确的文本 （加删除线的文本）  |
|  `<samp>` | 计算机代码样本 |
| `<script>`  | 定义客户端脚本 |
| `<section>`  | 文档中的一个部分 |
| `<select>`  | 下拉列表 |
| `<small>`  | 小号文本 |
|  `<source>` | 为媒体元素（`<video>` 和 `<audio>`）定义多个媒体资源  |
| `<span>`  | 文档中的节 |
| `<strike>`  | 删除线文本 （使用 `<del>` or `<s>` 代替）|
| `<strong>`  | 重要（强调）文本 |
| `<style>`  | 文档的样式信息 |
| `<sub>`  | 下标文本 |
| `<summary>`  | 为 `<details>` 元素定义可见标题  |
| `<sup>`  | 上标文本 |
| `<svg>`  | SVG 图形容器|
| `<table>`  | 表格 |
|  `<tbody>` | 表格中的主体内容 |
|  `<td>` | 表格中的单元格 |
| `<template>`  | 为页面加载时应隐藏的内容定义一个容器  |
| `<textarea>`  | 多行输入控件（文本区域） |
| `<tfoot>`  | 表格中的表注内容（脚注）|
| `<th>`   | 表格中的标题单元格 |
| `<thead>`  | 表格中的表头内容 |
|`<time>`   | 特定时间（或日期时间）|
| `<title>`  | 文档的标题 |
| `<tr>`  | 表格中的一行 |
| `<track>`  | 媒体元素的文本轨道（`<video>` 和 `<audio>`）|
| `<tt>`  | 电传文本 （使用CSS代替）|
|  `<u>`  | 下划线文本 |
| `<ul>`  | 无序列表 |
| `<var>`  | 变量 |
| `<video>`  | 嵌入式视频内容 |
| `<wbr>`  | 规定在文本中的何处适合添加换行符  |
